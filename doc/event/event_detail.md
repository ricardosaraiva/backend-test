# Informações do evento
Retorna as informações do evento

* **URL**

  /event/:id/detail

* **Method:**

  `GET`
*  **URL Params**

   **Required:**
   `id=[integer]`

* **Success Response:**

  * **Code:** 200 <br />
    **Content:** `{
	"id": 1,
	"name": "xxx",
	"description": "xxxx",
	"date": "0000-00-00 00:00:00",
	"city": "xxxxx",
	"state": "xx",
	"address": "xxxxx, xxxx",
	"cancel": false
}`
